Expert Note¶

Expert: 朱有亮

这是一个动态共价键驱动的聚合反应过程模拟，体系初始由一种单体组成，单体由四个粗粒化粒子组成，中心粒子A为惰性粒子，三个连接粒子B为活性粒子，单体之间的两个B粒子可以产生动态共价键，成键后形成一个B-B键、两个B-B-B键角，其中B最多只能反应一次，成键的能垒Ebar，断键的能垒为Ebar+Ebind，成键判断的截断半径为2的 (1.0 / 6.0)次方，为了减小能量波动，成键形成的B-B-B的最小角度为120度，成键判断的周期是2000步，由于要形成的键和键角初始的时候体系没有，因此需要调用all_info.addBondTypeByPairs()和all_info.addAngleTypeByPairs()这两个函数，在设置相应的相互作用之前，添加到系统中。断键模块由于是独立模块，需要设置和体系相同的键和键角相互作用，为了计算能量的时候使用，用于判断断键条件。

runDynamics.py¶

 1from poetry import cu_gala as gala
 2from poetry import _options
 3filename = 'pn2d.xml'
 4build_method = gala.XMLReader(filename)
 5perform_config = gala.PerformConfig(_options.gpu)
 6all_info = gala.AllInfo(build_method, perform_config)
 7dt = 0.0005
 8app = gala.Application(all_info, dt)
 9neighbor_list = gala.NeighborList(all_info, 2 ** (1.0 / 6.0), 0.1)
10lj = gala.LJForce(all_info, neighbor_list, 2 ** (1.0 / 6.0))
11lj.setParams('A', 'A', 1.0, 1.0, 1.0)
12lj.setParams('B', 'B', 1.0, 1.0, 1.0)
13lj.setParams('A', 'B', 1.0, 1.0, 1.0)
14app.add(lj)
15all_info.addBondTypeByPairs()
16all_info.addAngleTypeByPairs()
17bf = gala.BondForceHarmonic(all_info)
18bf.setParams('A-B', 1250.0, 1.0)
19bf.setParams('B-B', 1250.0, 1.0)
20app.add(bf)
21af = gala.AngleForceHarmonic(all_info)
22af.setParams('B-A-B', 1000.0, 120)
23af.setParams('A-B-B', 1000.0, 180)
24app.add(af)
25group = gala.ParticleSet(all_info, 'all')
26comp_info = gala.ComputeInfo(all_info, group)
27T = 1.0
28bd = gala.LangevinNVT(all_info, group, T, 123)
29app.add(bd)
30sort_method = gala.Sort(all_info)
31sort_method.setPeriod(100)
32app.add(sort_method)
33comp_info = gala.ComputeInfo(all_info, group)
34DInfo = gala.DumpInfo(all_info, comp_info, 'data.log')
35DInfo.setPeriod(200)
36app.add(DInfo)
37xml = gala.XMLDump(all_info, 'p')
38xml.setPeriod(100000)
39xml.setOutputMass(True)
40xml.setOutputImage(True)
41xml.setOutputBond(True)
42xml.setOutputAngle(True)
43xml.setOutputInit(True)
44xml.setOutputCris(True)
45app.add(xml)
46app.run(10000)
47Ebar = 2.77
48Ebind = 6.0
49rever = gala.DePolymerization(all_info, T, 16361)
50rever.setParams('B-B', 1250.0, 1.0, 1.0, 1000.0, 180.0, Ebar + Ebind, 1.0, gala.DePolyFunc.harmonic)
51rever.setPeriod(2000)
52rever.setCountUnbonds(100000)
53app.add(rever)
54reaction = gala.Polymerization(all_info, neighbor_list, 2 ** (1.0 / 6.0), 123)
55reaction.setEnergyBar(Ebar)
56reaction.setPr('B', 'B', 1.0)
57reaction.setNewBondTypeByPairs()
58reaction.setNewAngleTypeByPairs()
59reaction.generateAngle(True)
60reaction.setMaxCris('B', 1)
61reaction.setInitInitReaction(True)
62reaction.setAngleLowerLimitDegree(120)
63reaction.setPeriod(2000)
64app.add(reaction)
65app.run(5000000)
66neighbor_list.printStats()
